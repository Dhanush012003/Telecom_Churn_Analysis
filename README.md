**📊 Customer Churn Analysis & Prediction**

**🚀 Project Overview**

This project focuses on analyzing customer churn behavior and predicting customers who are likely to leave a service.
The goal is to help businesses identify at-risk customers and take proactive actions to improve retention.

**🎯 Objectives**
Analyze customer demographics and behavior
Identify key factors influencing churn
Build a predictive model for churn detection
Visualize insights using interactive dashboards

**📌 Key Insights**
📉 Overall Churn Rate: 27%
👥 Total Customers: 6,418
⚠️ Predicted Churners: 378 customers
💡 Higher churn observed in:
Month-to-month contracts
Fiber optic users
Customers with higher monthly charges

**📊 Dashboard Features**
🔹 Churn Summary Dashboard
Total Customers, New Joiners, Churn Rate
Churn by Gender, Age Group, and Tenure
Churn Rate by:
Contract Type
Payment Method
Internet Service Type
State-wise churn analysis
Churn reason distribution
🔹 Churn Prediction Dashboard
Predicted churn customer count
Gender distribution of churners
State-wise customer distribution
Risk segmentation by:
Age
Tenure
Payment method
Detailed table of high-risk customers

**🛠️ Tools & Technologies Used**
SQL – Data extraction & transformation
Python – Data preprocessing & model building
Power BI – Data visualization & dashboards
Excel – Data cleaning & analysis

**🤖 Machine Learning Model**
Model Used: (Random Forest)
Features considered:
Monthly Charges
Tenure
Contract Type
Payment Method
Internet Service
Output:
Churn Probability
Customer Risk Classification

**📈 Business Impact**
Helps reduce churn by identifying high-risk customers
Enables targeted marketing strategies
Improves customer retention and revenue
